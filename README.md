# dbdes

database description
对数据库的描述，实际用途是简化，查询表结构、及相关表字段的含义。  
  

只需要通过关键字，即可查询相关表，及相关字段。  

增加根据表，生成java模板代码bean dao 等  




## 技术栈
- jdk1.8
- spring boot
- mybatis
- bootstrap
- themeleaf
- mysql数据库
- Redis
